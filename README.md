# Expense Tracker

A simple command-line expense tracker written in Python.

## Features

- Add expenses
- View expenses
- Edit expenses
- Delete expenses
- View total spending
- View spending by category
- Monthly spending summary
- Highest and lowest expense

## Installation

Clone the repository:

git clone https://github.com/naylinhtut20/expense-tracker.git

Create virtual environment:

python -m venv .venv

Activate it:

.venv\Scripts\activate

Run the program:

python expense_tracker.py
